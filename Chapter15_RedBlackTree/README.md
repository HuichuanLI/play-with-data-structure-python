### 时间复杂度
- 查找的话 还是 AVL 快
- 添加/删除的话 还是红黑树


![](../photo/27.png)


![](../photo/28.png)


    对于完全随机的数据，普通的二分搜索树很好用！ 
    缺点：极端情况退化成链表（或者高度不平衡） 
    对于查询较多的使用情况，AVL树很好用！
    红黑树牺牲了平衡性（2logn的高度）
